# SharePoint skills

Store SharePoint skill markdown files in this folder.

Use one subfolder per skill, for example:

`sharepoint/<skill-name>/`

Current skills include:

### Content and Compliance

- `content-expiration-sentinel/` — identifies outdated or unmanaged content by analyzing activity, ownership, and content signals
- `process-compliance-first-pass/` — reviews documents against required compliance elements before formal review

### Content Publishing and Dashboards

- `excel-to-html-dashboard/` — converts Excel workbooks into branded HTML dashboards

### Knowledge and Navigation

- `library-destination-advisor/` — recommends the right SharePoint document library for uploaded content
- `onboarding-path-synthesizer/` — builds role-based onboarding paths from SharePoint content
