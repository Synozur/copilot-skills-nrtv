# Narrative Strategies — Deck Brand Kit (extracted from client pitch deck)

Source: client-provided `Narrative Strategies Pitch Deck Short.pptx`. Brand elements
only — colors, fonts, and logos. The original deck's specific slide layouts are NOT
reproduced; this kit applies the brand to clean slides built from scratch.

## Slide format
16:9 widescreen — 13.333 in × 7.5 in (the deck's native size).

## Typography (Montserrat family)
The deck's theme nominally lists Aptos, but the slides themselves are set in **Montserrat**
(96 runs) — that is the real brand face. Use:

| Role | Typeface |
|------|----------|
| Title / headline | Montserrat SemiBold |
| Section / slide heading | Montserrat SemiBold |
| Body / bullets | Montserrat |
| Light captions | Montserrat Light |

Montserrat is a Google font; if absent on the opening machine, PowerPoint substitutes the
nearest sans-serif. Brand colors and logos are unaffected.

## Color palette (as used on the slides)

| Token | Hex | Use |
|-------|-----|-----|
| Navy (primary)   | `184579` | title/section backgrounds, headings, the dominant brand color |
| Blue (bright)    | `135897` | secondary headings, links, accents |
| Pale steel       | `C8CFDC` | dividers, subtle fills, captions on navy |
| Slate            | `577199` | secondary text, chart accents |
| Gold (accent)    | `FFC755` | sparing highlight — rules, key numbers, call-outs |
| Cyan (accent)    | `37BFDA` | sparing highlight — secondary accent |
| White            | `FFFFFF` | text on navy, page background |
| Near-black       | `202124` | body text on white |

Use gold and cyan sparingly as accents only — navy + white is the backbone.

## Logos (in `assets/`)

| File | Use |
|------|-----|
| `narrative-wordmark-white.png` | White "N" mark + NARRATIVE wordmark — for **navy / photo / dark** slides (title, section dividers). |
| `narrative-wordmark-navy.png`  | Navy wordmark — for **white / light** content slides. |
| `narrative-mark-navy.png`      | Standalone navy "N" mark — small footer/corner accent on light slides. |

Place the white wordmark only on dark backgrounds and the navy wordmark/mark only on
light backgrounds. No other recoloring.
